# Coffee Landing Page — Bean Scene

Responsive landing page for a coffee shop brand. Built with semantic HTML and CSS.

**Live Demo:** *(enable GitHub Pages to publish)*  
**Author:** [annavoy](https://github.com/annavoy)

---

## About

Single-page marketing site with hero, menu, about, features, and contact sections. Includes a mobile burger menu and custom typography.

## Features

- Responsive layout for mobile, tablet, and desktop
- Navigation with anchor links
- Menu cards with product images
- Mobile burger menu
- Custom Google Fonts styling

## Tech Stack

- HTML5
- CSS3 (Flexbox, media queries)
- SVG icons

## Run Locally

```bash
git clone https://github.com/annavoy/coffe-landing-page.git
cd coffe-landing-page
open index.html
```

## Deploy to GitHub Pages

1. Repo **Settings → Pages**
2. Source: **Deploy from a branch** → `main` → `/ (root)`
3. Live at: `https://annavoy.github.io/coffe-landing-page/`
